# Ερωτηματολόγιο Επιλογής Ομαδικού Έργου (ΠΛΗΠΡΟ)

Μια απλή εφαρμογή για τη συλλογή προτιμήσεων και την επιλογή ομαδικού έργου με δημοκρατικό τρόπο (Ranked Voting).

## Χαρακτηριστικά
- **Ranked Voting:** 1η (3π), 2η (2π), 3η (1π) επιλογή.
- **Veto:** Δυνατότητα απόρριψης έργου.
- **Admin Dashboard:** Αυτόματος υπολογισμός αποτελεσμάτων και στατιστικών.

## Εγκατάσταση & Εκτέλεση (Τοπικά)

1. Κλωνοποίηση ή λήψη του φακέλου.
2. Εγκατάσταση εξαρτήσεων:
   ```bash
   npm install
   ```
3. Εκκίνηση του server:
   ```bash
   npm start
   ```
4. Άνοιγμα στον browser:
   - Ψηφοφορία: `http://localhost:3000`
   - Αποτελέσματα: `http://localhost:3000/admin.html`

## Δεδομένα
Τα δεδομένα αποθηκεύονται προσωρινά στη μνήμη του server (RAM). Αν γίνει restart, χάνονται.

## Deployment (Vercel)
Το project είναι έτοιμο για deploy στο Vercel. Απλά συνδέστε το repo και πατήστε Deploy.
